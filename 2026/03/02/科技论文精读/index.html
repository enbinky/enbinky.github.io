Pseudodata-based molecular structure generator to reveal unknown chemicals

基于伪数据的分子结构生成器以揭示未知化学物质

一、核心研究对象

本研究聚焦暴露组学（Exposomics） 中的核心科学问题 ——将质谱数据直接转化为化学分子结构，实现环境中未知多氟烷基和全氟烷基物质（PFAS，全氟化合物）的快速识别与发现，具体研究对象包括：

  1. 多氟烷基 / 全氟烷基物质（PFAS）：这类物质广泛存在于环境中，具有神经毒性、生殖和代谢干扰等危害，且大量替代物为未公开的商业秘密，是当前环境暴露组学的重点研究难点；
  2. 质谱 - 分子结构转化模型（MSGo）：本研究开发的基于伪数据和 Transformer 的分子结构生成器，是研究的核心载体；
  3. 质谱数据（MS1/MS2）：包括实验质谱、通过 CFM-ID 生成的虚拟伪质谱数据，是模型的核心输入；
  4. 拓展验证对象：脂质分子（验证模型策略的通用性）。

研究同时针对现有方法的痛点：实验质谱数据稀缺、分子结构空间庞大、未知化合物人工鉴定效率低、现有模型性能受限等问题展开。

二、核心研究方法

本研究的整体思路是构建伪质谱数据集解决数据稀缺问题→设计概率导向掩码策略优化伪数据质量→改造 Transformer 神经网络搭建 MSGo 模型→多维度验证模型性能→将模型应用于实际环境样本发现未知 PFAS→验证策略对其他分子的通用性，所有方法均为可复现设计，核心模块及关键参数如下（复现重点标注）：

（一）数据集构建：分子库 + 质谱库（实验 + 虚拟）

为解决 PFAS 实验质谱数据稀缺的问题，研究构建了分子结构库和质谱库（实验 + 虚拟伪数据），这是复现的基础步骤：

  1. PFAS 分子结构库

    • 原始收集 42159 个 PFAS 分子，经 RDKit 标准化 SMILES、去除立体化学信息、有机金属化合物和重复项后，最终保留33942 个 PFAS 分子的 SMILES（复现核心数据集，需从论文补充数据 / GitHub 获取）；
    • 采用SMILES 配对编码（SmilesPE） 将 SMILES 转化为亚结构令牌序列，提取 821 个高频亚结构令牌，序列长度设为 30。

  2. 实验质谱库

    • 从商业数据库（SCIEX）、公共库（NORMAN、Massbank）、自建库和 2019-2021 年文献中收集 367 个 PFAS 电喷雾电离模式的实验质谱，去重后保留300 个实验质谱（对应 219 个 PFAS 分子），并按置信度分为 3 级（1 级 131 个、2 级 88 个、3 级 81 个）；
    • 仅保留质谱的m/z信息，构建m/z词汇表（保留出现超 5 次的m/z，添加母离子m/z），序列长度固定为 25。

  3. 虚拟伪质谱库（核心复现点）

    • 采用CFM-ID v3.0为 33942 个 PFAS 分子生成虚拟质谱，涵盖低（10V）、中（20V）、高（40V）三种碰撞能量，最终得到101826 个虚拟质谱，解决实验数据稀缺问题；
    • 质谱质量精度验证了 1 Da、0.1 Da、0.01 Da 三个梯度，最终选择最优参数用于模型训练。

（二）概率导向掩码策略：伪数据优化的核心（复现关键）

由于虚拟质谱与实验质谱差异大（89% 的配对相似度 < 0.5），研究设计概率导向掩码策略降低二者差异、提升伪数据可靠性，这是 MSGo 性能的核心保障：

  1. 掩码原理：移除虚拟质谱中低丰度（易出错）的碎片离子，保留噪声离子，让模型学习区分真实离子和噪声；
  2. 掩码概率计算（核心公式）：
  3. 掩码参数：质谱整体掩码概率为 50%，m/z的掩码概率由其丰度决定，每个训练轮次（epoch）对同一化学物质的虚拟质谱重新掩码，避免过拟合；
  4. 优化效果：掩码后虚拟 - 实验质谱相似度 > 0.5 的配对占比从 11% 提升至 32%。

（三）MSGo 模型架构：改造的 Transformer 神经网络（复现核心）

MSGo 基于经典 Transformer（Attention is all you need）改造，适配质谱→分子结构的转化任务，移除了编码器的位置编码（因质谱碎片顺序无意义），添加了解码器的分子式编码，核心架构和参数如下（复现需严格匹配）：

  1. 输入层：编码器输入为处理后的质谱m/z序列（母离子 + 掩码后碎片离子），解码器输入为 PFAS 的分子式编码 + 亚结构令牌序列；
  2. 网络结构：6 层编码器 + 6 层解码器，8 个注意力头，嵌入维度 512，前馈层隐藏维度 2048；
  3. 激活与输出：激活函数为 ReLU，Softmax 计算令牌概率分布，输出为符合语法的 SMILES 序列；
  4. 生成策略：采用基于亚结构的生成策略（而非逐字符生成），提升结构预测准确性；通过束搜索（Beam Search） 捕获全局最优 SMILES，经多次验证将num_beams 设为 500（平衡计算资源和准确率，num_beams=1 为贪心搜索，性能显著降低）。

（四）模型训练与优化（复现关键参数）

MSGo 基于 Python 3.7+PyTorch 实现，训练硬件为 Intel Core i7-11700k+GeForce RTX 3090 Ti GPU（Linux 环境），核心训练参数如下（复现需严格遵循）：

  1. 数据集划分：33942 个 PFAS 分子 + 虚拟质谱随机分为训练集（31942）、验证集（1000）、测试集（1000）；
  2. 优化器与学习率：ADAM 优化器，学习率 0.0004；
  3. 训练轮次与批次：200 个 epoch，批次大小（batch size）=50；
  4. 正则化：dropout=0.1%，采用交叉熵作为损失函数，训练过程无过拟合；
  5. 训练效率：单模型训练时间约 6 小时；
  6. SMILES 有效性：生成的 SMILES 有效率达 95.3%，确保语法正确性。

（五）多维度性能验证方法

为验证 MSGo 的性能，研究设计了实验室验证（实验质谱）、对比验证（与现有工具）、实际样本验证（环境废水） 三层验证体系，复现时可参考该验证逻辑：

  1. 实验室验证：用 300 个实验质谱测试，统计Top K 正确识别率（K=1/3/5/10），核心指标：Top1 正确识别率 48.0%，Top10 达 80.7%；同时定义有效结构（含相同官能团 / 亚结构的异构体），用于评估未知化合物的预测能力；

  2. 对比验证：

    • 与代谢组学主流工具对比：随机选 100 个实验质谱，对比CFM-ID v3.0、SIRIUS v5.8，MSGo 准确率显著高于二者；
    • 与现有分子生成模型对比：复现MSNovelist、MassGenie、Spec2Mol、MS2Mol并在同一数据集上训练，MSGo 的注释准确率和结构相似度均最优；
    • 验证指标：SMILES 正确生成率、基于ECFP/FCFP 分子指纹的 Tanimoto 系数（衡量生成结构与目标结构的相似度）；

  3. 实际样本验证：采用氟化工园区废水的质谱数据（已由专家人工鉴定），直接输入 MSGo（无降噪处理），对比 MSGo 与专家、MZcloud 数据库的识别结果，验证模型在复杂环境样本中的实用性。

（六）实际样本应用：发现未知 PFAS

将 MSGo 应用于氟化工园区废水的质谱数据，完成已知 PFAS 验证和未知 PFAS 发现：

  1. 对专家鉴定的 15 类 90 个 PFAS，MSGo Top1 正确预测 11 类 31 个，Top10 有效预测 12 类 43 个；
  2. 发现6 类新的 PFAS，其中二氯取代全氟烷基羧酸盐（Cl₂-PFCAs） 和氯 / 氢取代全氟辛烷磺酸盐（H,Cl-PFOS） 为环境暴露组学中首次发现，且 Cl₂-PFCAs 未包含在训练集中，验证了模型的泛化能力和未知化合物发现能力；
  3. 从解离路径、同位素分布和专利文献验证了新 PFAS 的合理性，证明其实际存在可能性。

（七）通用性验证：脂质分子拓展

为验证伪数据 + 掩码策略的通用性，研究将 MSGo 架构迁移至脂质分子：

  1. 收集 > 30000 个脂质分子，用 CFM-ID 生成虚拟质谱，按相同参数训练模型；
  2. 验证结果：Top1 正确识别率 27.7%，Top10 达 50.3%，性能优于现有模型；
  3. 结论：模型性能与分子结构空间匹配度相关，脂质结构多样性更高，增加训练集规模可进一步提升性能。

三、复现实验的核心注意事项

  1. 数据获取：从论文指定的GitHub（http://github.com/aaronma2020/MSGO） 和Zenodo（https://doi.org/10.5281/zenodo.17182996） 获取 33942 个 PFAS 分子的 SMILES、300 个实验质谱、虚拟质谱生成脚本，这是复现的基础；
  2. 工具版本：严格匹配实验工具版本 ——CFM-ID v3.0、SIRIUS v5.8、RDKit 2020.09.1.0、PyTorch 1.7.1、Python 3.7，避免版本兼容问题；
  3. 关键参数：掩码策略的概率公式、Transformer 的架构参数（6 层编解码、嵌入维度 512）、束搜索 num_beams=500、训练的 200 个 epoch 和 ADAM 学习率 0.0004 是 MSGo 性能的核心，不可随意修改；
  4. 硬件要求：建议使用至少 1 块 RTX 3090 Ti/4090 或同等算力 GPU，Linux 环境可提升训练效率；（我要哭死了，呜呜呜）
  5. 验证逻辑：先完成实验室小规模验证（用 100 个实验质谱测试），再对比现有工具，最后应用于实际样本，逐步验证模型性能；
  6. SMILES 处理：必须用 SmilesPE 进行亚结构令牌化，而非逐字符处理，这是提升模型准确性的关键。

四、研究的核心创新点

  1. 提出伪数据驱动的训练策略，用 CFM-ID 生成虚拟质谱解决暴露组学中实验质谱数据稀缺的问题；
  2. 设计概率导向掩码策略，实现虚拟质谱与实验质谱的域适配，提升伪数据可靠性，是 MSGo 性能的核心；
  3. 改造 Transformer 神经网络，移除编码器位置编码、添加解码器分子式编码，并采用亚结构基生成策略，适配质谱→分子结构的转化任务；
  4. 开发的 MSGo 首次实现了暴露组学中未知 PFAS 的自动化快速识别，性能优于现有工具，还发现了 2 类全新的环境 PFAS；
  5. 验证了伪数据 + 掩码策略的通用性，为其他大类化合物（如脂质、代谢物）的质谱 - 结构转化提供了可复用的范式。
