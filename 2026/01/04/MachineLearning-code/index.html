库

numpy：专门做数值计算，提供高效数组、矩阵运算，是所有数据库的底层基础。用于高效的数值计算，是几乎所有科学计算库的基础。

Pandas：用于数据清洗、分析和处理，操作数据表格（DataFrame）的利器。

matplotlib：Python 最基础的绘图库，能画折线图、柱状图、散点图等所有基础图表。

seaborn：基于 Matplotlib 美化版绘图库，画统计图表更美观、代码更简单。

sklearn（Scikit-learn）：专门做机器学习，封装了分类、回归、聚类、降维等所有经典算法。

机器学习算法

K-近邻算法（KNN）

一种非参数化、懒惰学习的监督学习算法，可用于分类和回归任务。

对一个新样本，找到训练集中距离最近的K个邻居，根据这些邻居的类别或数值来预测新样本的类别或数值。

源代码+未知知识学习

1.np.array().reshape()

  • .reshape()：是 NumPy 数组（ndarray）的内置方法，用于改变数组的形状（维度），但不改变数组中元素的总数和元素本身。

  • 参数详解：

    第一个参数-1：这是 NumPy 中reshape方法的特殊占位符，含义是 “让 NumPy 自动计算该维度的大小”，计算的依据是 “数组总元素数不变”。

    第二个参数1：明确指定数组的第二维（列数）为 1

2.numpy的布尔索引

机器学习二分类数据可视化的经典写法，核心难点是 numpy 的布尔索引

plt.scatter(x, y)

绘制二维散点图，必须传入两组数据：x 轴坐标 + y 轴坐标

X_train[y_train == 0, 0] → x 轴坐标

这是 numpy 高级索引（核心！）

  • y_train == 0：筛选出所有标签 = 0 的样本行
  • 后面的 0：取这些样本的第 0 列特征（作为散点的 x 坐标）

color='blue'：类别 0 的点用蓝色

label='Class 0'：图例显示 “类别 0”
